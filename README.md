# Personal Portfolio

A personal portfolio website built using **Next.js with TypeScript**, focused on performance, clean design, and maintainable code structure.

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons / Lucide
- Vercel

---

## Features

- Server-side rendering and SEO optimization
- Fully responsive layout
- Clean and minimal UI
- Component-based architecture
- Optimized images with Next.js Image
- Smooth animations
